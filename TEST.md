# MeQR 手動テスト手順書

本ドキュメントは、[MeQR](https://yumakakuya.github.io/meqr/) の動作確認のための手動テスト手順です。

---

## 1. テストデータの投入

### 方法A: メニューから読み込む

1. https://yumakakuya.github.io/meqr/ を開く
2. ハンバーガーメニュー（☰）を開く
3. **「データの保存・削除」** をタップ
4. **「テストデータを読み込む」** をタップ
5. 確認ダイアログで **OK** を選択

→ ビジネス用・プライベート用の2枚のサンプル名刺が作成されます。

### 方法B: URLパラメータで自動読み込み

1. ブラウザで次のURLを開く  
   **https://yumakakuya.github.io/meqr/?demo=1**
2. 表示された確認ダイアログで **OK** を選択

→ 同上、テストデータが読み込まれます。

## 2. 表示・編集の確認

| # | 確認項目 | 手順 | 期待結果 |
|---|----------|------|----------|
| 2-1 | 表示モード | 「👁️表示モード」が選択されている | 名前・組織・役職・連絡先が表示され、編集フォームは非表示 |
| 2-2 | 編集モード | 「✏️編集モード」をタップ | フォームが表示され、入力・保存ボタンが使える |
| 2-3 | 名刺切り替え | 画面上部の名刺セレクターで「プライベート用」を選択 | 2枚目の名刺の内容に切り替わる |
| 2-4 | QRコード | 表示モードでQRコードが表示されている | サイズ設定（大・中・小）が反映されている |

## 3. 保存・共有・エクスポート

| # | 確認項目 | 手順 | 期待結果 |
|---|----------|------|----------|
| 3-1 | 保存してQR更新 | 編集で名前などを変更 → 「保存してQRを更新」 | 表示・QRが更新され、ローカル保存される |
| 3-2 | .vcf ダウンロード | 「💾 .vcf」をタップ | vCard ファイルがダウンロードされる |
| 3-3 | QR保存 | 「⬇️ QR保存」で PNG または SVG を選択 | QRコード画像がダウンロードされる |
| 3-4 | 共有（vCard） | 「📤 共有」をタップ | Web Share API で共有画面が出る（未対応環境ではフォールバック） |
| 3-5 | vCard確認 | 「🔎 確認」をタップ | vCard プレビューが表示される |
