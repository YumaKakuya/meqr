# MeQR

プライバシー重視のデジタル名刺アプリ。QRコードで連絡先を共有し、データは端末内のみに保存します。

## 名刺作成コンセプト（統合ドキュメント）

別プロジェクトで検討していた名刺作成アプリ「メイシット」のコンセプト・ワイヤーフレームを MeQR に統合しています。

- **[仕様書（確定仕様書）](docs/specification.html)** — 機能要件・テンプレート・QR・出力・課金・技術要件・データ構造・ユーザーフロー・開発スケジュール
- **[ワイヤーフレーム v2](docs/wireframe-v2.html)** — 職業選択→おすすめテンプレート→編集→プレビュー→印刷設定の画面構成と v1/v2 比較

アプリ内では **メニュー → このアプリについて** から同じドキュメントへアクセスできます。

## 技術

- 単一 HTML（PWA）
- オフライン対応・ローカルストレージ完結
- vCard / MECARD 出力・QRコード生成

## ライセンス

オープンソース（リポジトリのライセンスに準拠）。
